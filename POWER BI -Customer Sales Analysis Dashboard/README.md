# Customer Sales Analysis Dashboard

## Objective
Develop an interactive Power BI dashboard using Microsoft Excel as the data source to analyze customer demographics, account balances, regional distribution, and job type segmentation. The dashboard provides insights into customer behavior, financial trends, and regional performance to support business decision-making.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key Features
- Customer Balance Analysis by Age and Gender
- Customer Distribution by Age Group
- Regional Balance Analysis
- Customer Segmentation by Job Type
- KPI Cards (Maximum Balance, Minimum Balance, Gender Analysis)
- Customer Trend Analysis Over Time

## DAX Measures Used
- Maximum Balance
- Minimum Balance
- Customer Count
- Total Balance

## Business Insights
- Analyzed customer balances across different age groups and genders.
- Identified customer distribution by region and job type.
- Monitored maximum and minimum account balances.
- Tracked customer trends over time.
- Enabled interactive analysis using filters and visualizations.
