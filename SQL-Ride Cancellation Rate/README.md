# Ride Cancellation Analysis

## Objective
Analyze daily ride cancellation rates for non-banned users between specific dates using SQL.

## Tools Used
- SQL

## SQL Concepts Used
- JOIN
- CASE
- GROUP BY
- COUNT
- ROUND
- Aggregate Functions

## Business Problem
Calculate the daily cancellation rate by considering only rides where both the client and driver are not banned.

## Business Insights
- Calculated the cancellation rate for each day.
- Excluded banned users from the analysis.
- Measured daily service performance using cancellation rate.

## Skills Demonstrated
- SQL Query Writing
- Data Analysis
- Business Analytics
- KPI Calculation
