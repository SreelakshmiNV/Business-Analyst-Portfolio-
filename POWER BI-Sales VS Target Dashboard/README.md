# Sales vs Target Dashboard

## Objective
Develop an interactive Power BI dashboard using Microsoft Excel as the data source to compare actual sales with sales targets, evaluate team and salesperson performance, analyze monthly sales trends, and identify performance gaps to support business decision-making.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key Features
- KPI Cards (Total Sales, Sales Target, Variance, Variance %)
- Sales vs Target Comparison
- Individual Salesperson Performance
- Team-wise Performance Analysis
- Monthly Sales Trend Analysis
- Interactive Dashboard

## DAX Measures Used
- Total Sales Actual
- Total Sales Target
- Variance
- Variance %

## Business Insights
- Compared actual sales with target sales.
- Identified top-performing salespeople and teams.
- Analyzed monthly sales trends.
- Measured the sales gap using variance and variance percentage.
- Supported performance monitoring through interactive visualizations.
