# HR Analytics Dashboard

## Objective
Develop an interactive Tableau dashboard using Microsoft Excel as the data source to analyze employee attrition, workforce demographics, job roles, promotions, and employee experience. The dashboard uses interactive filters to support HR decision-making and employee retention strategies.

## Tools Used
- Tableau
- Microsoft Excel

## Dataset
HR Employee Dataset

## Key Features
- Employee Attrition Analysis
- Attrition by Age
- Attrition by Monthly Income
- Attrition by Job Role
- Promotion vs Current Role
- Years at Company Analysis
- Working Years Analysis
- Department Filter
- Interactive Dashboard

## Business Insights
- Identified employee attrition trends across different age groups.
- Compared attrition among different job roles.
- Analyzed the impact of monthly income on employee attrition.
- Evaluated employee experience and promotion trends.
- Supported HR decision-making through interactive visualizations.
