# HR Interactive Excel Dashboard

## 📌 Objective

The objective of this project is to develop an interactive HR dashboard using Microsoft Excel to analyze employee data, monitor workforce performance, evaluate recruitment and salary trends, track employee demographics, and provide actionable insights for HR decision-making through dynamic visualizations and interactive reporting.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting
- Data Validation

---

## ✨ Features

- Interactive HR dashboard with KPI Cards
- Employee demographics analysis
- Department-wise employee distribution
- Salary analysis and comparison
- Recruitment source analysis
- Employee performance analysis
- Employee engagement analysis
- Attendance and absenteeism analysis
- Dynamic slicers and filters for interactive reporting

---

## 📊 Business Insights

- Identified workforce distribution across departments and job roles.
- Compared active and terminated employees to evaluate workforce stability.
- Analyzed salary trends to support compensation planning.
- Evaluated recruitment sources contributing to employee hiring.
- Monitored employee performance and engagement levels.
- Identified absenteeism and attendance patterns to support HR planning.
- Enabled data-driven HR decisions through interactive dashboard reporting.
