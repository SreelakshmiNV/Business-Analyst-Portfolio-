# UK Bank Customer Analysis Dashboard

## Objective
Develop an interactive Tableau dashboard using Microsoft Excel as the data source to analyze customer demographics, regional distribution, account balances, gender, age groups, and job classifications. The dashboard uses interactive filters to support customer segmentation and banking business decisions.

## Tools Used
- Tableau
- Microsoft Excel

## Dataset
UK Bank Customer Dataset

## Key Features
- Customer Distribution by Region
- Customer Distribution by Gender
- Customer Distribution by Age Group
- Balance Distribution Analysis
- Job Classification Analysis
- Interactive Filters
- Interactive Dashboard

## Business Insights
- Analyzed customer distribution across different UK regions.
- Compared customer demographics by age and gender.
- Evaluated account balance distribution.
- Identified customer segments based on job classification.
- Supported customer profiling through interactive dashboards.
