# Boston Housing Analytics Dashboard

## Objective
Develop an interactive Tableau dashboard using Microsoft Excel as the data source to analyze crime rate, housing prices, and property characteristics. The dashboard uses interactive filters to identify factors affecting housing values and supports data-driven real estate analysis.

## Tools Used
- Tableau
- Microsoft Excel

## Dataset
Boston Housing Dataset

## Key Features
- Crime Rate vs Property Price
- Crime Rate vs NOX
- Crime Rate vs TAX
- Crime Rate vs RAD
- Crime Rate vs LSTAT
- Housing Feature Description
- Interactive Dashboard

## Business Insights
- Identified the relationship between crime rate and housing prices.
- Compared environmental factors affecting property values.
- Analyzed the impact of taxation and accessibility on crime rates.
- Supported housing market analysis using interactive dashboards.
