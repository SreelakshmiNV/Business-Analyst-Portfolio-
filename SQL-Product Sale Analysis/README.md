# Product Sales Analysis

## Objective
Analyze product sales data to calculate total revenue generated by each product using SQL.

## Tools Used
- SQL

## SQL Concepts Used
- ALTER TABLE
- UPDATE
- JOIN
- SUM
- GROUP BY
- ORDER BY

## Business Problem
Calculate the total revenue for each product and rank products based on revenue generated.

## Business Insights
- Identified the highest revenue-generating products.
- Calculated total revenue using quantity and price.
- Ranked products in descending order of revenue.

## Skills Demonstrated
- SQL Query Writing
- Data Analysis
- Revenue Analysis
- Business Analytics
