# Maven Market Sales Dashboard

## Objective
Develop an interactive sales dashboard using Microsoft Excel as the data source and Power BI for visualization to analyze sales performance, profitability, customer returns, and transactions. The dashboard provides key business insights and supports data-driven decision-making.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key Features
- KPI Cards (Total Profit, Total Returns, Current Month Transactions)
- Product Performance Analysis
- Revenue Trend Analysis
- Geographic Sales Analysis
- Revenue vs Target Comparison
- Interactive Slicers and Filters

## DAX Measures Used
- Total Profit
- Total Returns
- Current Month Transactions
- Profit Margin
- Return Rate
- Revenue vs Target

## Business Insights
- Monitored overall sales performance and profitability.
- Identified top-performing products and locations.
- Compared revenue against business targets.
- Analyzed weekly revenue trends and return rates.
- Enabled interactive filtering for better business analysis.

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX Measures
- Data Visualization
